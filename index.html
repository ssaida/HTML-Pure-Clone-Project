When did HTML first page ever written?

The first web page using HTML was created by Tim Berners-Lee, the inventor of the World Wide Web, in 1991. The web page was served from a NeXT computer at CERN, a European research organization, and it contained basic information about the World Wide Web project. The web page was simple, with black text on a white background and hyperlinks to other pages. It was the beginning of a revolution in information sharing and communication that continues to this day.
